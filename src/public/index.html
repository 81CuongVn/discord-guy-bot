GuyBot

A friendly Discord bot that helps server members use more inclusive language.

Invite to Discord

Gracious to Start

When a user sends words like "guys", "lads", "chaps", etc. in a message, a gentle DM is sent to the user reminding them to use inclusive language.

Louder for Those in the Back

Subsequent uses of "guys", "lads", "chaps", etc. will warn the user for all to see in the channel.
